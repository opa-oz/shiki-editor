# shiki-packages
This repository contains some core shikimori js libraries: `shiki-editor`, `shiki-uploader`, `shiki-utils` and `shiki-decorators`.

## Demo
1. Clone this repository
2. Start demo
```sh
yarn
cd demo
yarn
yarn serve
```
